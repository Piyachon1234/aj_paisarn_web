## โครงสร้างระบบ
- frontend: React.js
- backend: Node.js
- Database: MongoDb
- model

## Required Software
- VS code(for working on code) or other IDE
- Docker for running localhost
## Requirement
- anaconda environment
- npm >=v18

## วิธีรันบนDocker
1. `docker compose build` (หากมีการแก้ไขโค้ด)
2. `docker compose up -d`

## วิธีรันผ่านVS code
1. `npm start`
